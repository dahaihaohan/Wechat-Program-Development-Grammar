
一、CSS基本语法
	CSS语句是由两部分组成的：选择器和属性声明
		1.选择器决定了这个样式渲染那些标签或者组件元素
		2.被花括号括起来的部分叫做属性声明，属性声明部分是CSS语句的主体，
		  决定了被选择器选择的标签或者组件元素采用怎样的方式来渲染。
		3.一组属性声明分为两部分：属性名（key）和属性值（value）。
		  属性名和属性值之间用冒号（：）间隔，每一组属性声明都要以分号（；）结束
		4.声明的语句块要以花括号（{}）括起来
二、选择器
	.class:
		{
			样例：.intro
			样例描述：选择所有拥有class="intro"的组件
		}
	#id:
		{
			样例：#firstname
			样例描述：选择拥有id="firstname"的组件
		}
	element:
		{
			样例：view
			样例描述：选择所有view组件
		}
	element,element:
		{
			样例：view,checkbox
			样例描述：选择所有文档的view组件和所有checkbox组件
		}
	::after:
		{
			样例：view::after
			样例描述：在view组件后边插入内容
		}
	::before:
		{
			样例：view::before
			样例描述：在view组件前边插入内容
		}
三、id选择器
		以“#”号开头的选择器都是ID选择器
		例如，下面的样式表规则会对属性id=“p1”的组件或标签起作用
			#p1{
				color:#333;
				text-align:left;
			}
		注意：有些Web浏览器支持id属性以数字开头，但微信小程序不支持这样的规则
四、class选择器
		如果要对一组组件或者标签使用样式表，就要使用class选择器
			如所有的view元素使用class="col"让元素的颜色统一起来
				view.col{
					color:#aaa;
				}
		与id选择器类似，在小程序中，class选择器类名的第一个字符不能使用数字
五、使用CSS样式表
		内联样式：
			语法格式：<组件名 style="属性1：值1;属性2：值2;....">
		外部样式表：
			支持用@import语句导入外联样式表文件，@import后跟需要导入的外联样式表的相对路径
六、WXML新增特性
		rpx和px：描述尺寸单位
		同时也支持CSS3标准新引入的rem尺寸单位
		微信规定所有的屏幕宽度都是20rem，所以rem与rpx的换算关系是1rem=(750/20)rpx
